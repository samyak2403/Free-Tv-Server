## ğŸ“‹ Pull Request Process

1. **Fork & Clone**
   ```bash
   git clone https://github.com/YOUR_USERNAME/servercontrol.git
   cd servercontrol
   ```

2. **Create Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Changes**
   - Write clear code
   - Add comments
   - Test thoroughly

4. **Commit**
   ```bash
   git add .
   git commit -m "Add: clear description of changes"
   ```

5. **Push**
   ```bash
   git push origin feature/your-feature-name
   ```

6. **Open PR**
   - Go to GitHub
   - Click "New Pull Request"
   - Describe your changes
   - Link any related issues
